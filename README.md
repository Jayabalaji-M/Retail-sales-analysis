# 📊 Retail Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing retail sales data to understand business performance across different categories, customer segments, and time periods.

The dashboard was built using **SQL Server** for data processing and **Power BI** for visualization, enabling interactive exploration of key business metrics.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing product categories
- Understand customer behavior across age groups
- Track revenue trends over time
- Provide insights for data-driven decision-making

---

## 🛠️ Tools & Technologies
- **SQL Server** – Data cleaning and querying  
- **Power BI** – Dashboard creation and visualization  

---

## 📂 Dataset
- Dataset contains **2000+ records**
- Includes:
  - Sales transactions
  - Product categories
  - Customer demographics (age, gender)
  - Order details and timestamps

---

## 📊 Key Metrics
- **Total Revenue:** 912K  
- **Total Orders:** 1997  
- **Average Order Value:** 456.54  
- **Total Profit:** 721.96K  
- **Profit Margin:** 79.19%  

---

## 📈 Key Insights
- Clothing category recorded the highest quantity sold  
- Electronics generated the highest revenue (314K)  
- Senior age group contributed the most revenue (451K)  
- Evening shift had the highest number of orders (~1.3K)  
- Revenue showed noticeable peaks during late 2022 and late 2023  

---

## 📊 Dashboard Features
- Interactive filters:
  - Category (Beauty, Clothing, Electronics)
  - Shift (Morning, Afternoon, Evening)
  - Gender
  - Date range  
- KPI cards for quick business overview  
- Visuals for trend analysis and comparisons
